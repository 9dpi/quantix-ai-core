MT4 Bridge Installation Guide (V1.2 - 2026 Edition)

This guide provides step-by-step instructions to connect your MetaTrader 4 terminal to our institutional AI signals network.

Broker Verification: This system is Broker-Agnostic. It works with any MT4 broker (IC Markets, Pepperstone, XM, etc.). Ensure you are logged into your specific broker's server before activating the EA.
 MAC MINI / MACOS USERS
Compatibility Note Running MT4 on Apple Silicon (M1/M2/M3)

The Signal Genius EA is fully optimized for Mac Mini and MacBook users. However, since MT4 is natively a Windows app, please follow these rules:

  • Recommended Software: Use the official MT4 version provided by your broker (which uses a Wine-wrapper) or PlayOnMac.
  • Rosetta 2: Ensure Rosetta 2 is installed on your Mac Mini for the wrapper to execute correctly.
  • Data Folders: When opening "Data Folder" on Mac, it will open inside a virtual Windows drive (e.g., drive_c). This is normal.
  • WebRequest Stability: Mac wrappers can be stricter with network permissions. Ensure Auto-Trading is toggled ON at the top of MT4.

1. Broker Server Login

Requirement Select Your Specific Broker Server

Do not use the default "MetaQuotes" demo. Search for your specific broker name.

  • Go to File -> Login to Trade Account
  • Search for your broker (e.g., Pepperstone) and select the designated server.
  • Ensure the connection bars in the bottom right turn green/blue.

2. File Placement

Step One Deploy Expert Advisor
  • Open MT4 Terminal -> File -> Open Data Folder
  • Navigate to: MQL4 > Experts
  • Copy Signal_Genius.mq4 into this folder
  • In MT4 Navigator, right-click Expert Advisors and Refresh

3. Terminal Configuration

Step Two Whitelist Production URL
  • Go to Tools -> Options (or Cmd+O on Mac)
  • Select the Expert Advisors tab
  • Check: [X] Allow automated trading
  • Check: [X] Allow WebRequest for listed URL:
Add exactly this address:
https://quantixapiserver-production.up.railway.app/

4. Activation

Step Three Attach to Chart
  • Open any M15 chart (e.g., EURUSD)
  • Drag Signal_Genius from Navigator onto the chart
  • In Inputs:
    - ApiToken: Use your license key
    - SymbolSuffix: (e.g., .r or .pro if needed)
  • In Common: Check [X] Allow live trading
Success: Look for the "Smiley Face" in the top-right corner of your chart. If it is smiling, the bridge is active. Check the Experts tab for "EA Polling API..." logs.

5. Risk Management

Adaptive Scaling Irfan-Standard Risk Model
  • RiskPercent: Default 2.0 (2% of balance per trade).
  • LotScalingMode: DYNAMIC_EQUITY ensures lots grow as your balance grows.
