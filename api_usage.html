Market Intelligence API

Quantix developer API allows you to integrate institutional-grade market structure reasoning into your own execution engines and platforms. We provide the context, you provide the action.

⚠️ Responsible Usage Policy

Quantix API does not provide Buy/Sell signals, Stop Loss, or Take Profit levels. Users are strictly prohibited from marketing Quantix as a signal service. This API provides deterministic market state analysis for decision support only.

Base URL

https://quantixaicore-production.up.railway.app/api/v1

Endpoints

GET /public/market-state

Retrieves the current structural state, confidence, and reasoning evidence for a specific asset.

Parameters

Parameter Type Required Description
asset String Yes Symbol (e.g. EURUSD, GBPUSD)
timeframe String No H1, H4, or D1 (Default: H4)

Authentication

Include the following header in your request:

X-API-Key: YOUR_API_KEY

Sample Response

{
  "asset": "EURUSD",
  "timeframe": "H4",
  "state": "bullish",
  "confidence": 0.7842,
  "dominance": {
    "bullish": 1.42,
    "bearish": 0.31
  },
  "evidence": [
    "Bullish BOS confirmed (82% acceptance)",
    "No fakeout detected after swing break",
    "Higher low structural integrity maintained"
  ],
  "updated_at": "2026-01-15T04:00:00Z",
  "trace_id": "struct-9e21c",
  "disclaimer": "Quantix AI provides research insights only. Not financial advice."
}

Integration Guidance

As a developer, you should map the state and confidence to your own trading bias. Common deterministic mappings:

LONG_BIAS
state == "bullish" AND confidence > 0.75
SHORT_BIAS
state == "bearish" AND confidence > 0.75
NO_BIAS
state == "range" OR confidence < 0.5
