MT4 Bridge Installation Guide (V1.0)

This guide provides step-by-step instructions to connect your MetaTrader 4 terminal to our institutional AI signals network.

Broker Verification: This system is Broker-Agnostic. It works with any MT4 broker (IC Markets, Pepperstone, XM, etc.). Ensure you are logged into your specific broker's server (e.g., PepperstoneUK-Demo03) before activating the EA.

0. Critical: Broker Server Login

Requirement Select Your Specific Broker Server

Mobile/Desktop MT4 must be connected to your actual broker account, not the default "MetaQuotes" demo:

  • Go to File -> Login to Trade Account
  • In the Server field, do not use the default. Search for your specific broker name.
  • Example: Search for Pepperstone and select PepperstoneUK-Demo03.
  • Ensure the connection bars in the bottom right turn green/blue.

1. File Placement

Step One Deploy Expert Advisor
  • Open MT4 Terminal -> File -> Open Data Folder
  • Navigate to: MQL4 > Experts
  • Copy the Signal_Genius.mq4 file into this folder
  • In MT4, right-click Expert Advisors and select Refresh

2. Terminal Configuration

Step Two Whitelist Production URL

To communicate with our AI servers, yours needs permission:

  • Go to Tools -> Options (or Ctrl+O)
  • Select the Expert Advisors tab
  • Check: [X] Allow automated trading
  • Check: [X] Allow WebRequest for listed URL:
Add exactly this address:
https://quantixapiserver-production.up.railway.app/

3. Activation

Step Three Attach to Chart
  • Open any chart (e.g., EURUSD, M15)
  • Drag Signal_Genius from Navigator onto the chart
  • In Inputs tab, verify:
    - ApiUrl: https://quantixapiserver-production.up.railway.app/api/v1/mt4/signals/pending
    - ApiToken / License Key: DEMO_MT4_TOKEN_2026
  • In Common tab, check [X] Allow live trading
Verification: Check the "Experts" tab at the bottom. You should see "EA Polling API: https://..." and no Error 5200 messages. This confirms a live bridge.

4. Dynamic Lot Scaling

Adaptive Risk Management Account Growth Scaling

The system uses an institutional grade Risk % Model. Lot sizes are calculated to ensure you only risk a specific percentage of your balance per trade.

  • UsePercentageRisk: Set to true to enable balance-based scaling.
  • RiskPercent: Default 1.0 (1% risk per trade). This is the "Irfan Standard".
  • SymbolSuffix: Always ensure this is set to .r (for Pepperstone) or your broker's specific suffix.

5. Advanced Configuration

Customization How to Change Risk & Targets
A. Change Risk % (e.g., 2% like Irfan)

Managed directly in MT4:

  • Press F7 on the chart.
  • Change Risk % per trade from 1.0 to 2.0.
  • Click OK. Applies immediately to next trade.
B. Change TP/SL (e.g., 7p/12p)

Managed in the Cloud Backend:

  • Open backend/quantix_core/config/settings.py
  • Find TP_PIPS and SL_PIPS.
  • Update values (e.g., 7.0 and 12.0).
  • Save. The AI brain updates instantly.
