1. System Mandate
Quantix AI has transitioned from a predictive model to an Auditable Reasoning Engine. The primary goal is determining market state (Structure, Trend, Regime) with mathematical certainty.

2. Data Law (Clean Feed v1)
Dukascopy tick-level data is ingested using BID-only pricing. All candles are UTC-aligned. Data integrity is ensured by a FAIL-HARD validator that rejects any partial or corrupted candles.

3. Structure Resolution
The Structure Engine resolves "Swing Points", "BOS" (Break of Structure), and "CHoCH" (Change of Character) deterministically. Every state resolution can be copied as a Trace ID and audited back to raw tick data.
