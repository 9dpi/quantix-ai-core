1. System Infrastructure
Quantix AI Core is a distributed probabilistic engine. It consists of a FastAPI backend (Railway), a Supabase PostgreSQL database, and a minimalist frontend.

2. Data Pipeline
- Drop CSV files in the Ingestion page.
- Use download_eurusd_data.py to fetch Dukascopy or Yahoo Finance data.
- The Quality Gate filters non-tradable candles (low volume/volatility).

3. Core Intelligence
The engine matches patterns across historical market memory to generate high-probability sniper alerts. "Alpha Weight" represents the adjusted confidence based on past outcome feedback.
